# uber-eats
Курс ["Веб-разработка для начинающих: HTML и CSS"](https://stepik.org/course/38218/info).
 
Раздел 9. [Деплой сайта](https://stepik.org/lesson/242296/step/1?unit=214732).
 
## Выпускной проект
Согласно
[макету в Фигме](https://www.figma.com/file/8lxQ3PGYTHQsCgTXnEJre8/Uber-Eats?node-id=0%3A1) 
сверстать сайт _Uber Eats_, состоящий из двух страниц – **списка ресторанов** и **страницы ресторана**.
Страницы связаны друг с другом ссылками – клик по любому ресторану из списка переносит на страницу 
с рестораном «Пушкин».
 
 - Необходимо воспользоваться **CSS-сеткой** `flexboxgrid2`.
 [Документация по flexboxgrid2](https://evgenyrodionov.github.io/flexboxgrid2/).
 
 - Использовать три основных экрана-брейкпоинта: `Desktop`, `Tablet` и `Mobile`.
     
 - Для каждого из этих брейкпоинтов есть макеты страниц.
 Обе страницы должны адаптироваться под разную ширину экрана.
